Case Study

StudentHub - University ERP System

A full-stack university management system for students, faculty, courses, and administrative operations, built with Spring Boot, React.js, MySQL, Spring Security, and JWT.

Back to portfolio 
StudentHub {
  APIs: "Spring Boot REST",
  Auth: "Spring Security + JWT",
  Data: "MySQL + JPA/Hibernate",
  Outcome: "Role-aware ERP workflow"
}

Problem

Academic operations need organized, secure workflows.

Goal

Create a dashboard for managing student, faculty, course, and administrative records with authenticated access.

Backend

Built RESTful APIs using Spring Boot and connected them with database-backed CRUD operations.

Security

Implemented user authentication and role-based authorization using Spring Security and JWT.

Outcome

Created a role-aware ERP foundation that organizes academic records and demonstrates end-to-end full-stack delivery.
